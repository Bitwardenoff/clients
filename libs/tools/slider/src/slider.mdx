# Slider

This component is custom styled `input[type=range]` (ie. slider) input to allow users to choose a
numeric value by manipulating a thumb along a track with a defined min and max values. It includes
an accompying number field for keyboard input. Keyboard input via left and right arrow is also
supported for the slider when the element is in focus.
